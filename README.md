# Knight's Travails

A JavaScript implementation of the Knight's Travails problem, finding the shortest path for a knight to move from one square to another on a chessboard using **Breadth-First Search (BFS)** on a graph data structure.

# Features

- Finds shortest path (minimum moves)
- Validates inputs (ensures squares are on 8×8 board)
- Handles edge cases (same start/end, legal moves)
- Clean path output

JavaScript (Logic)
